# Public Folder

This folder contains static assets that will be served directly by Next.js.

## Common files to add here:

- **favicon.ico** - Website favicon
- **images/** - Image assets (logos, photos, etc.)
- **icons/** - Icon files
- **robots.txt** - Search engine crawler instructions
- **sitemap.xml** - XML sitemap for SEO

Files in this folder can be referenced directly from the root URL, e.g.:
- `/favicon.ico` → `public/favicon.ico`
- `/images/logo.png` → `public/images/logo.png`
